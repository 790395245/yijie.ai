# 周易AI - UI/UX设计文档

## 当前状态分析

### 已有功能
1. 六爻起卦（随机/时间）
2. 卦象可视化展示
3. AI解读功能
4. 流畅的动画效果
5. API配置（Claude/OpenAI）

### 当前问题
1. 所有功能集中在一个页面，信息密度高
2. 缺少引导流程，新用户可能不知道如何使用
3. API配置暴露在主界面，影响美观
4. 缺少历史记录功能
5. 缺少帮助说明

## 设计目标

1. **简洁优雅**：主界面聚焦核心功能
2. **引导清晰**：新用户能快速上手
3. **体验流畅**：操作流程自然顺畅
4. **神秘感**：符合周易占卜的氛围
5. **现代化**：融合传统与现代设计

## 改进方案

### 1. 页面布局优化

**主界面简化**
- 隐藏API配置到设置面板
- 突出起卦按钮
- 添加简短的引导文案

**设置面板**
- 右上角设置图标
- 侧边栏滑出式设计
- 包含API配置、语言切换、帮助说明

### 2. 交互流程设计

**起卦流程**
```
1. 用户进入 → 看到简洁的主界面
2. 点击起卦按钮 → 动画效果展示起卦过程
3. 显示卦象 → 自动展开卦象详情
4. 点击"获取解读" → AI流式输出解读
5. 查看完整解读 → 可保存到历史记录
```

**首次使用引导**
- 检测是否首次访问
- 显示简短的引导提示
- 介绍核心功能

### 3. 视觉设计优化

**色彩方案**
- 主色：紫色系（神秘、智慧）
- 辅色：金色/粉色（高贵、柔和）
- 背景：深色渐变（营造氛围）

**图标和装饰**
- 使用中国传统元素
- 八卦符号装饰
- 流动的粒子效果

### 4. 功能增强

**历史记录**
- 保存占卜历史
- 快速查看过往卦象
- 支持导出分享

**帮助系统**
- 六爻基础知识
- 使用说明
- 常见问题
